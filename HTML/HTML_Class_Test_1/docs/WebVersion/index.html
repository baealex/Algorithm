데비안 패키지 설치

먼저 컴파일을 할 수 있는 패키지를 설치해 주도록 하겠습니다.

sudo apt-get install build-essential

위 명령어를 입력하여 패키지를 설치합니다. 위 명령어로 패키지를 설치하면 gcc, g++, make, perl등의 기본적이고 가장 필수적인 라이브러리가 설치됩니다.
