**[문제2] 리스트의 더하기와 extend**

다음과 같은 리스트 a가 있다.

```
>>> a = [1, 2, 3]
```

리스트 a에 [4, 5]를 + 기호를 이용하여 더한 결과는 다음과 같다.

```
>>> a = [1, 2, 3]
>>> a = a + [4,5]
>>> a
[1, 2, 3, 4, 5]
```

리스트 a에 [4,5]를 extend를 이용하여 더한 결과는 다음과 같다.

```
>>> a = [1, 2, 3]
>>> a.extend([4, 5])
>>> a
[1, 2, 3, 4, 5]
```

기호를 이용하여 더한것과 extend한 것의 차이점이 있을까? 있다면 그 차이점에 대해서 얘기해 보자.
